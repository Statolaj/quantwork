# quantwork
A repository dedicated to simple programming tasks relevant for me in computational finance.

The current list of files includes:
  1) dice_game.py which implements an optimal strategy for stopping a dice game where you roll a six-sided die for a fixed number of rounds. The goal is maximizing the expected number of pips. Uses recursion. 
  2) gaussian_moments.py which simulates the standard Gaussian $m^{th}$ moment using Monte Carlo. The files contains 5 different version: A) Simple Monte Carlo, B) Importance Sampling from variance $m+1$, C) Importance Sampling from variance $m+1$ and antithetic variables, D) Chi^2 proposals and antithetic variables, E) Mixture of two Gaussian at $\pm \sqrt{m}$ and antithetic variables. The best performing version is D, where essentially the variance is 0 of the estimator.
  3) heat_equation_solver.py which implements a finite difference method for solving the heat equation with a boundary condition in dimensions 1 and 2. The method can be extended to higher dimension, but since the number of grid points grows exponentially in the dimension, it is not possible to apply this method in practice. Instead use Feynmann-Kac to and simulate the expectation using Monte Carlo, where the precision does not depend on the dimension.  
  4) pi_estimation.cpp which implements a simple Monte Carlo estimation of pi in C++ using uniform samples from a 2x2 square and computing the fraction that land inside the unit circle. With uniform samples, the acceptance rate is pi/4 = 78.5%, but an idea to improve this is using importance sampling and switching to e.g. 2*Beta(2,2) so generally points are closer to the center.
